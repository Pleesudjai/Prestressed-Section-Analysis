1.5  Optimized Prestress Force — from Magnel Diagram
------------------------------------------------------------------
  The Magnel diagram gives 1/F as a function of eccentricity e.
  At the actual tendon CG eccentricity e_cg(x) at each section,
  the program reads the governing lower bound of 1/F (which equals
  1/F_min). The required force at each section is F_min = 1/(1/F_min).

  Governing F_min values (from feasibility4Sections.m):
    Section 1  (x =  0 ft):  F_min =    0.0 kip   [no moment]
    Section 2  (x = 10 ft):  F_min =  352.0 kip
    Section 3  (x = 20 ft):  F_min =  424.4 kip   [upper bound: F_max]
    Section 4  (x = 32 ft):  F_min =  424.5 kip   [GOVERNING]

  F_req = max(F_min all sections) = 424.5 kip   [computed, not input]

  For construction: n = ceil(424.5 / 25.0) = 17 strands  -->  F_prov = 425.0 kip
  The kern-point analysis below uses the exact optimized F_req = 424.5 kip.

3.1  At Transfer  (f'ci = 4.8 ksi)

  Tension (ACI 318-19 Sec. 24.5.3.2):

             = -6 x sqrt(6000) / 1000
             = -6 x 77.46 / 1000

  NOTE: sigma_ti and sigma_ts are NEGATIVE (tension sign).

        Naaman's equations use these SIGNED values directly.


==================================================================
   PART 4 — KERN-POINT QUANTITIES  (Naaman Notation)
==================================================================

  PURPOSE:
    The kern-point equations solve for the FEASIBLE ECCENTRICITY
    RANGE [e_min, e_max] at a given prestress force F_i and
    moment demand M. They use the same 4 stress conditions as the
    Magnel diagram but with e_o as the unknown (F is fixed).

  NAAMAN KERN-POINT DEFINITIONS:
    k_b = +St / Ac  = +r^2 / y_t  (lower kern, POSITIVE, below centroid)
    k_t = -Sb / Ac  = -r^2 / y_b  (upper kern, NEGATIVE, above centroid)
    Z_t = St  (section modulus, top fiber)
    Z_b = Sb  (section modulus, bottom fiber)

  COMPUTATION:
  ------------------------------------------------------------------
    k_b = +St / Ac = +4535.1 / 487.0 = +9.312 in

    k_t = -Sb / Ac = -1701.1 / 487.0 = -3.493 in

  NOTE ON KERN SIGN:
    k_b > 0 because the lower kern is BELOW the centroid.
    k_t < 0 because the upper kern is ABOVE the centroid.
    A positive eccentricity e > 0 means the tendon is BELOW the
    centroid (consistent with compression-positive convention).

  GOVERNING EQUATIONS  (Naaman Table 4.2, Row 2):
  ------------------------------------------------------------------
    Line I  : e_o <= k_b + (1/F_i)(M_min - sigma_ti x Z_t)    [UPPER bound]
              --> Transfer, top fiber:  f_top >= sigma_ti

    Line II : e_o <= k_t + (1/F_i)(M_min + sigma_ci x Z_b)    [UPPER bound]
              --> Transfer, bot fiber:  f_bot <= sigma_ci

    Line III: e_o >= k_b + [1/(eta x F_i)](M_max - sigma_cs x Z_t)  [LOWER bound]
              --> Service,  top fiber:  f_top <= sigma_cs

    Line IV : e_o >= k_t + [1/(eta x F_i)](M_max + sigma_ts x Z_b)  [LOWER bound]
              --> Service,  bot fiber:  f_bot >= sigma_ts

  FEASIBLE RANGE:
    e_max = min(Line I,   Line II)    (e_o must be <= both upper bounds)
    e_min = max(Line III, Line IV)    (e_o must be >= both lower bounds)
    Feasible if and only if  e_min <= e_max

  NOTE on sigma_ti and (- sigma_ti):
    sigma_ti = -0.2078 ksi  (already negative)
    (M_min - sigma_ti x Z_t) = M_min - (-0.2078)(4535.1)
                              = M_min + 942.6 kip-in
    The MINUS sign before sigma_ti converts it to an ADD because
    sigma_ti is itself negative (tension convention).
