5.1  Derivation of the Four Lines  (Naaman notation)
------------------------------------------------------------------
  The Magnel diagram plots 1/F vs eccentricity e.
  F   = resultant prestress force (all strands combined).
  lam = 1/F

  STRESS EQUATIONS (simply supported, compression positive):
    At Transfer (eta = 1.0, moment = Mi):
      f_top = +F/Ac - F*e/St + Mi/St
      f_bot = +F/Ac + F*e/Sb - Mi/Sb

    At Service (effective force = eta*F, moment = MT):
      f_top = +eta*F/Ac - eta*F*e/St + MT/St
      f_bot = +eta*F/Ac + eta*F*e/Sb - MT/Sb

  LINE I  — Top fiber @ Transfer, tension limit (lower bound on 1/F):
    Requirement:  f_top >= -f_ti_allow  (keep tension above limit)
    Rearranged:   1/F >= (e/St - 1/Ac) / dI
    where dI  = f_ti_allow + Mi/St

  LINE II — Bot fiber @ Transfer, compression limit (lower bound):
    Requirement:  f_bot <= +f_ci_allow
    Rearranged:   1/F >= (1/Ac + e/Sb) / dII
    where dII = f_ci_allow + Mi/Sb

  LINE III — Bot fiber @ Service, tension limit (UPPER bound on 1/F):
    Requirement:  f_bot >= -f_ts_allow
    Rearranged:   1/F <= eta * (1/Ac + e/Sb) / dIII
    where dIII = MT/Sb - f_ts_allow     [requires dIII > 0]
    => Governs F_min because a higher F (smaller 1/F) is needed
       to counteract large service tension at the bottom fiber.

  LINE IV — Top fiber @ Service, compression limit (lower bound):
    Requirement:  f_top <= +f_cs_allow
    Rearranged:   1/F >= eta * (1/Ac - e/St) / dIV
    where dIV = f_cs_allow - MT/St

  FEASIBILITY CONDITION:
    max(lineI, lineII, lineIV) <= 1/F <= lineIII
    and  1/F > 0
    and  e_geo_min <= e <= e_geo_max

------------------------------------------------------------------
  Sections 3 and 4 share the same e_cg (both tendons at y=6 in),
  but their F bounds nearly cross:

    Section 3 requires:  F <= 424.4 kip   (Line II, bot compr @ transfer)
    Section 4 requires:  F >= 424.5 kip   (Line III, bot tens @ service)

  The gap is only 0.1 kip, but it means no single F satisfies
  BOTH sections exactly at this eccentricity profile.

  The feasible F range for the full beam (intersection across all

    F must be >= 424.5 kip  (Section 4 governs minimum)
    F must be <= 424.4 kip  (Section 3 governs maximum)
    => No F satisfies all constraints simultaneously.

7.3  Engineering Resolution
------------------------------------------------------------------
  With discrete strand increments of 25 kip:
    16 strands: F = 400 kip  -- midspan service tension FAILS
    17 strands: F = 425 kip  -- drape point transfer compression
                                 marginally exceeds limit by 0.006 ksi

  Verify F = 425 kip at Section 3 (x=20 ft, e=14.362 in):
    Mi at x=20 ft = 0.04227*240*(768-240)/2 = 2678.5 kip-in

    f_bot @ transfer = +425/487.0 + 425*14.362/1701.1 - 2678.5/1701.1
                     = +0.8726 + 3.5870 - 1.5745
                     = +2.886 ksi
    Limit: +2.880 ksi  --> EXCEED by 0.006 ksi (0.2%)

  The exceedance of 0.006 ksi (0.2%) is within typical engineering
  tolerance and rounding of material properties. For a class
  project this is acceptable. In practice, one would:
    Option A: Accept 0.2% marginal exceedance (engineering judgment)
    Option B: Debond 1 strand at x=20 ft to slightly reduce e_cg
              at the drape point while keeping midspan e unchanged

  --> Design decision: Use 17 strands (F = 425 kip), noting the
      marginal 0.2% exceedance at the drape point (x=20 ft).
